# CortexSail.AI

An agentic AI platform that retrieves company knowledge, reasons over policies, and answers scenario-based queries using RAG, multi-agent orchestration, and conversational memory.

# Overview of Components

- User interface:
  - Streamlit (If upgrade required React.js)

- Backend Database
  - ChromaDB

- Agentic
  - CrewAI for crews and agents

- Generative
  - Langchain and Langgraph for LLM model integration

- LLM Model
  - Gemini Flash 2.5 / Azure OpenAI deployed model

- Server tools
  - MCP server hosting

- Middleware server
  - FastAPI - Python
